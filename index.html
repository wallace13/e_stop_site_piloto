Sistema de Gerenciamento do Estacionamento



Sobre

E-STOP é um software desenvolvido para atender às necessidades de estacionamentos, que lidam diariamente com o armazenamento de veículos (tanto de clientes assinantes, quanto de clientes que utilizam o serviço rotativo), mas que encaram a dificuldade de organização dos registros de pagamento, cadastro de clientes, visualização de disponibilidade de vagas e as anotações de movimentação do caixa.

Objetivo

Facilitar o controle de vagas e controle de clientes assinantes de estacionamentos privados, bem como agilizar o processo de registro de entrada e saída de veículos.

Desenvolvimento

O E-STOP foi desenvolvido com base na metodologia ágil, utilizinado a linguagem de programação Java.

Benefícios

Redução do consumo de papel, mais rapidez no controle das vagas, segurança dos dados cadastrais e dos dados de operação de caixa.

Funcionalidades do sistema

Entrada e Saída

Mapeamento virtual das vagas ocupadas e disponíveis

Pagamento

Virtualização das receitas e despesas, dispensando o uso de papel.

Clientes

Cadastro digital de clientes, funcionários, veículos e assinaturas.

Relatório

Informatização dos registros.

Vagas

Reserva de vaga antecipadamente pelo cliente.

Segurança

Somente o gerente, possui acesso as informações financeiras.

Telas do Programa

Login

Tela inicial do sistema

Aba Principal

Local onde é possível: visualizar as vagas disponíveis e dar entrada ou saída de veículos.

Aba Cliente

Local onde é possível: Cadastrar um novo cliente ou consultar um já cadastrado.

Aba Caixa

Local onde é possível: Visualizar as movimentações financeiras, tal como fazer logout do sistema.

Aba Preços

Local onde é possível: Visualizar os preços da tabela do Rotativo, tal como os assinantes.

Aba Gerencia

Local onde é possível: Visualizar os usuários do sistema, atualizar os preços da tabela do Rotativo e os Planos.
